# Datavelopers Digest Archive

*dataveloper* (noun): a portmanteau of data and developer; a person who uses data and technology to build systems for real-world use.

---


**DATAVELOPERS Digest** is a human-curated (yours truly, [Chip Borodescu](https://www.linkedin.com/in/cborodescu/)) newsletter showcasing ideas, resources, and people shaping an ethical future through data and technology.

This repository contains the archive for the [Datavelopers Digest](https://www.linkedin.com/newsletters/datavelopers-digest-7194287769039388672/) newsletter on LinkedIn. Built or read something worth sharing with other datavelopers? Reach out at [chip@datavelopers.com](mailto:chip@datavelopers.com) and I'll add it to the weekly digest.
